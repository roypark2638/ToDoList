## Getting Started

### Installation

Clone the repo
   ```sh
   git clone https://github.com/roypark2638/ToDoList
   ```
   
### CocoaPods Install   
pod 'RealmSwift'
pod 'SwipeCellKit'
pod 'ChameleonFramework/Swift', :git => 'https://github.com/wowansm/Chameleon.git', :branch => 'swift5'
